ABOUT
Imgparagraph provides integration between Image,Text as Paragraph and add accordeon feature.

REQUIREMENTS
- Field_group module
- Paragraph module

INSTALLATION
Install the module as usual, more info can be found on:
http://drupal.org/documentation/install/modules-themeshttps://www.drupal.org/docs/8/extending-drupal-8/installing-drupal-8-modules

Enable Imgparagraph module under "Imgparagraph" package:
/admin/modules

USAGE / CONFIGURATION
Add paragraph field into any content type.
Create content of content type with paragraph field and select "Accordeon container" paragraph
as reference.
Create content and add "Accordeon container".
In "Accordeon container" select "accordeon type" left/right position of the image related to accordeon
Fill in fields of "accordeon image", "accordeon title", "accordeon content" with appropriate content.
Add another "Accordeon container". Fill those fields with the content.
Add as many "Accordeon container" as you like.
Save you content. Navigate to a newly created content to see working image accordeon.

P.S. "Accordeon container" paragraph as well as other necessary paragraphs and fields have to be created automatically while installing "Imgparagraph" module.
